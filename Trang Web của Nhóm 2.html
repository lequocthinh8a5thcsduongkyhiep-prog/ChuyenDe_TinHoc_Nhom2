CHUYÊN ĐỀ TIN HỌC

Làm quen với Visual Studio Code & Github

A. TỔNG QUAN

  • VS Code (Visual Studio Code): là trình soạn thảo mã nguồn miễn phí do Microsoft phát triển [cite: 4], hỗ trợ rất nhiều ngôn ngữ lập trình như C++, Python, Java, JavaScript… [cite: 4]; sở hữu hệ thống extension phong phú[cite: 5].
  • Github: là nền tảng lưu trữ và chia sẻ mã nguồn trực tuyến, dựa trên Git[cite: 6]. Nó cho phép quản lý phiên bản, làm việc nhóm, và tham gia vào các dự án mã nguồn mở[cite: 7].
  • Git: Git là hệ thống quản lý phiên bản (Version Control System) giúp theo dõi lịch sử thay đổi mã nguồn, tạo nhánh, hợp nhất và làm việc nhóm hiệu quả [cite: 8], do Linus Torvalds tạo ra[cite: 8].
  • GitHub Desktop: GitHub Desktop là ứng dụng giao diện (GUI) giúp bạn sử dụng Git và GitHub dễ dàng hơn mà không cần dùng lệnh[cite: 9].

B. CÀI ĐẶT & THIẾT LẬP

I. CÀI ĐẶT

1. VS Code

  1. Truy cập trang chính thức: https://code.visualstudio.com[cite: 13].
  2. Chọn nút **Download** phù hợp với hệ điều hành (Windows, macOS, Linux)[cite: 14].
  3. Mở file cài đặt → nhấn **Next** liên tục → chọn **Install**[cite: 15].
  4. Sau khi cài xong, mở VSCode để kiểm tra[cite: 16].
  5. (Khuyến nghị) Cài thêm extension[cite: 17]:
    • **GitLens:** Xem lịch sử commit, tác giả từng dòng[cite: 18].
    • **Prettier:** Định dạng mã nguồn tự động[cite: 19].
    • **GitHub Pull Requests & Issues:** Quản lý PR trực tiếp trong VS Code[cite: 20].

Lưu ý: Chỉnh cấu hình cơ bản (Theme, font, Tiếng Việt,.. nếu cần)[cite: 21].

2. Git

  1. Vào trang chính thức: https://git-scm.com/downloads[cite: 23].
  2. Chọn phiên bản phù hợp với hệ điều hành (Windows/macOS/Linux)[cite: 24].
  3. Mở file cài đặt → nhấn **Next** liên tục → **Install** → **Finish**[cite: 25].
  4. Kiểm tra bằng cách mở Terminal/Command Prompt và gõ:
    git --version
    [cite: 26]
  5. Nếu hiện số phiên bản là cài thành công[cite: 27].
  6. Cấu hình thông tin lần đầu[cite: 28]:
    git config --global user.name "Tên của bạn"
    [cite: 29]
    git config --global user.email "Email của bạn"
    [cite: 30]

Lưu ý: Các cấu hình thông tin sẽ gắn liền với các commit[cite: 31].

3. GitHub Desktop

  1. Truy cập https://desktop.github.com[cite: 33].
  2. Tải về và cài đặt như phần mềm bình thường[cite: 34].
  3. Đăng nhập bằng tài khoản GitHub[cite: 35].

Lưu ý:

  • Giao diện chính có các nút: **Clone a repository**, **Create a new repository**, **Add an existing repository**[cite: 37].
  • Có thể mở dự án bằng VSCode trực tiếp từ GitHub Desktop[cite: 38].

II. THIẾT LẬP

1. Tạo tài khoản GitHub

  1. Vào https://github.com[cite: 41].
  2. Nhấn **Sign up** → nhập email, mật khẩu, username[cite: 42].
  3. Xác nhận email để hoàn tất[cite: 43].

Lưu ý:

  • Sau khi đăng nhập, bạn có thể tạo repo mới bằng nút **New repository** (Tạo kho chứa mã nguồn mới)[cite: 45].
  • Chỉnh **Public** & **Private**[cite: 46]:
    • **Public:** Ai cũng xem được[cite: 47].
    • **Private:** Chỉ bạn (hoặc nhóm được cấp quyền) xem được[cite: 48].

2. Kết nối VS Code với GitHub

  1. Mở VSCode → nhấn **Ctrl + Shift + P**[cite: 50].
  2. Gõ: **GitHub: Sign in** → chọn đăng nhập bằng trình duyệt[cite: 51].
  3. Đăng nhập tài khoản GitHub → cấp quyền truy cập[cite: 52].
  4. VSCode sẽ lưu token xác thực → có thể push/pull trực tiếp[cite: 53].
  5. Kiểm tra tab **Source Control** đã hoạt động[cite: 54].

Lưu ý: Tab **Source Control** là nơi hiển thị mọi thay đổi[cite: 55].

3. Đăng nhập tài khoản GitHub trên GitHub desktop

  1. Mở GitHub Desktop[cite: 57].
  2. Nhấn **File** → **Options** (hoặc biểu tượng bánh răng)[cite: 58].
  3. Chọn **Sign in to GitHub.com**[cite: 59].
  4. Đăng nhập tài khoản → xác nhận[cite: 60].
  5. Sau khi đăng nhập, có thể clone, tạo repo, commit, push dễ dàng[cite: 61].

C. LÀM VIỆC CÙNG GITHUB

I. QUA VS CODE & GITHUB DESKTOP

1. Lấy dự án từ GitHub về để chỉnh sửa

a. Trực tiếp bằng VS Code
  1. Vào GitHub → chọn repo cần lấy[cite: 66].
  2. Nhấn nút **Code** → copy link **HTTPS**[cite: 67].
  3. Mở VSCode → Terminal → gõ:
    git clone <link_repo>
  4. Mở thư mục vừa clone → chỉnh sửa, commit, push[cite: 69].
b. Thông qua GitHub Destop
  1. Mở GitHub Desktop → chọn **Clone a repository** → chọn **URL**[cite: 71].
  2. Dán link repo hoặc chọn repo từ danh sách[cite: 72].
  3. Nhấn **Clone** → mở bằng VSCode để chỉnh sửa[cite: 73].

2. Đưa dự án của bản thân lên GitHub

a. Trực tiếp bằng VS Code
  1. Mở thư mục dự án bằng VSCode[cite: 76].
  2. Mở tab **Source Control** → nhấn **Initialize Repository**[cite: 77].
  3. Gõ nội dung commit → nhấn **Commit**[cite: 78].
  4. Nhấn **Publish to GitHub** → đăng nhập tài khoản[cite: 79].
  5. Chọn tạo repo mới hoặc repo có sẵn[cite: 80].
  6. Sau đó: **Commit** → **Push** mỗi khi cập nhật[cite: 81].
b. Thông qua GitHub Desktop
  1. Mở GitHub Desktop → chọn **Create a new repository**[cite: 83].
  2. Chọn thư mục dự án → đặt tên repo[cite: 84].
  3. Nhấn **Publish repository** để đưa lên GitHub[cite: 85].
  4. Sau khi chỉnh sửa → **Commit** → **Push origin**[cite: 86].
  5. Mở bằng VSCode để tiếp tục làm việc[cite: 87].

II. TRỰC TIẾP TRÊN GITHUB QUA GITHUB CODESPACE

1. GitHub CodeSpace là gì?

  • Là môi trường lập trình trực tuyến giống Visual Studio Code[cite: 90].
  • Chạy trực tiếp trên GitHub, không cần cài đặt phần mềm[cite: 91].
  • Tự động tải mã nguồn, cài thư viện, và lưu thay đổi[cite: 92].
  • Phù hợp để chỉnh sửa nhanh, làm việc nhóm, hoặc tham gia dự án mã nguồn mở[cite: 93].

2. Cách sử dụng GitHub CodeSpace

  1. Vào trang GitHub → đăng nhập tài khoản[cite: 95].
  2. Tìm địa chỉ repo bạn muốn chỉnh sửa (có thể là repo của người khác hoặc repo của bạn)[cite: 96].
  3. Dán địa chỉ repo vào thanh tìm kiếm → mở repo[cite: 97].
  4. Nhấn nút **Code** → chọn tab **Codespaces**[cite: 98].
  5. Nhấn **Create with Codespaces** hoặc **New codespace**[cite: 99].
  6. GitHub sẽ mở giao diện VSCode trực tiếp trong trình duyệt[cite: 100].
  7. Bạn có thể chỉnh sửa, commit, push ngay trên web mà không cần tải về[cite: 101].
  8. Nếu muốn đóng góp vào dự án người khác → tạo **Pull Request**[cite: 102].

D. THAM GIA DỰ ÁN MÃ NGUỒN MỞ

I. MÃ NGUỒN MỞ LÀ GÌ?

  • Là các dự án phần mềm có mã nguồn công khai, ai cũng có thể xem, sử dụng, chỉnh sửa và đóng góp[cite: 105].
  • Giúp cộng đồng cùng phát triển phần mềm, học hỏi lẫn nhau[cite: 106].
  • Các giấy phép phổ biến: **MIT**, **GPL**, **Apache**[cite: 107].

II. CÁCH THAM GIA VÀO DỰ ÁN MÃ NGUỒN MỞ

1. Tìm dự án phù hợp

  • Truy cập https://github.com/explore hoặc https://opensource.guide[cite: 110].
  • Tìm dự án theo ngôn ngữ bạn biết (Python, JavaScript, C++, v.v.)[cite: 111].
  • Ưu tiên chọn dự án có nhãn **good first issue** hoặc **help wanted**[cite: 112].

2. Quy trình tham gia

  1. **Fork** dự án: tạo bản sao repo về tài khoản của bạn[cite: 114].
  2. **Clone** repo fork về máy để chỉnh sửa:
    git clone <link_repo>
    [cite: 115]
  3. Tạo nhánh mới để làm việc riêng:
    git checkout -b ten-nhanh-moi
  4. Chỉnh sửa mã nguồn → **commit** → **push** lên repo fork[cite: 117].
  5. Tạo **Pull Request** để gửi thay đổi cho repo gốc[cite: 118].
  6. Chủ dự án sẽ **review** → nếu đồng ý sẽ **merge** vào dự án chính[cite: 119].

3. Lưu ý khi đóng góp

  • Bạn có thể thực hiện tương tự thông qua Github CodeSpace[cite: 121].
  • Đọc kỹ file **README.md** và **CONTRIBUTING.md** của dự án[cite: 122].
  • Viết **commit** rõ ràng, dễ hiểu[cite: 123].
  • Tôn trọng quy tắc cộng đồng, phản hồi lịch sự[cite: 124].
  • Có thể bắt đầu bằng việc sửa lỗi nhỏ, cải thiện tài liệu, hoặc thêm tính năng đơn giản[cite: 125].

E. TỔNG KẾT

  • **VS Code:** Công cụ viết code mạnh mẽ và phổ biến[cite: 127].
  • **Git:** Công cụ quản lý phiên bản không thể thiếu[cite: 128].
  • **GitHub:** Nơi lưu trữ, chia sẻ mã nguồn và làm việc nhóm[cite: 129].
  • **GitHub Desktop:** Dễ dùng, trực quan cho người mới[cite: 130].

→ Bộ công cụ giúp học sinh – sinh viên làm việc khoa học, chuyên nghiệp, hiệu quả hơn[cite: 131].
